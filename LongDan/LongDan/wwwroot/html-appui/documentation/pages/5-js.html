JavaScript

AppUI uses jQuery, Bootstrap JS framework as well as some other vital third-party jQuery plugins at its core. It also utilizes many third-party plugins which you can include per page if you need them.

Plugin Used on pages Description
Bootstrap All Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
jQuery All jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.
Bootstrap Colorpicker base_forms_pickers_select.html Fancy and customizable colorpicker plugin for Twitter Bootstrap.
Bootstrap Datepicker base_forms_pickers_select.html A datepicker for Twitter Bootstrap.
Bootstrap Notify base_ui_alerts_notify.html Turns standard Bootstrap alerts into "Growl-like" notifications.
Bootstrap Wizard base_forms_wizard.html This twitter bootstrap plugin builds a wizard out of a formatter tabbable structure.
Chartjs index.html
base_js_charts_chartjs.html		 Simple yet flexible JavaScript charting for designers & developers.
Datatables base_tables_datatables.html DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, based upon the foundations of progressive enhancement, and will add advanced interaction controls to any HTML table.
Dropzonejs base_forms_pickers_select.html DropzoneJS is an open source library that provides drag'n'drop file uploads with image previews.
Flot base_js_charts_flot.html Flot is a pure JavaScript plotting library for jQuery, with a focus on simple usage, attractive looks and interactive features.
gmaps.js base_js_maps.html gmaps.js allows you to use the potential of Google Maps in a simple way.
highlight.js base_js_syntax_highlight.html Syntax highlighting for the Web
jQuery Tags Input base_forms_pickers_select.html Magically convert a simple text input into a cool tag list with this jQuery plugin.

jQuery Validation base_forms_validation.html This jQuery plugin makes simple clientside form validation easy, whilst still offering plenty of customization options.
Magnific popup - Magnific Popup is a responsive lightbox & dialog script with focus on performance and providing best experience for user with any device.
Masked Input base_forms_pickers_select.html This plugin allows a user to more easily enter fixed width input where you would like them to enter the data in a certain format (dates,phone numbers, etc).
Select2 base_forms_pickers_select.html The jQuery replacement for select boxes.

Sparkline base_js_charts_sparkline.html This jQuery plugin generates sparklines (small inline charts) directly in the browser using data supplied either inline in the HTML, or via javascript.
